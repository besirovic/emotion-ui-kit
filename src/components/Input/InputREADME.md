# Input

## Props

| Prop name | Type     | Available options          | Default value |
| --------- | -------- | -------------------------- | ------------- |
| size      | `string` | `small`, `medium`, `large` | `medium`      |
| fullWidth | `bool`   | `true`, `false`            | `false`       |

## Code snippets

```js
  /** Example of text input with fullWidth and large size */
  <Input type="text" size="large" placeholder="Large text input" fullWidth  />
```

<br />

```js
  /** Example of password input */
  <Input type="password" placeholder="Your passowrd" />
```
